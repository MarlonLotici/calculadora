# Calculadora Simples

## Sobre o Projeto
Este é um projeto de calculadora funcional, construído como um portfólio para demonstrar habilidades fundamentais em desenvolvimento web.

## Funcionalidades
A calculadora realiza operações básicas de matemática:
- Adição (+)
- Subtração (-)
- Multiplicação (*)
- Divisão (/)
- Porcentagem (%)

## Tecnologias Utilizadas
- **HTML:** Para a estrutura dos botões e do display.
- **CSS:** Para a estilização e o layout visual da calculadora.
- **JavaScript:** Para toda a lógica de funcionamento e processamento dos cálculos.
